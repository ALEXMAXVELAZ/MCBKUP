# Minecraft Backup Manager

Small Python tool made to create safe backups of Minecraft Java Edition worlds.

## Features

- Automatic world detection from .minecraft/saves
- World size display (in MB)
- Timestamp-based backup naming
- Configurable backup folder
- Dark themed Tkinter UI
- Scrollable world list

## Why?

This project was created after losing important worlds.
The goal is to ensure backups are easy, fast, and reliable.

## Requirements

- Python 3.x
- Windows
- Minecraft Java Edition
